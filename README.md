
## Neo4j and rails 4 


### This is a training repo with the following stack:

    gem "neo4j", ">= 2.3.0"
    gem 'rails', '3.2.16'



### Installation


    git clone https://github.com/mrohon/neo4j-sample-app.git

    cd neo4j-sample-app

    bundle

    neo4j start

    rake db:seed

    rails s


### Online


   Application is deployed on Heroku [here](http://rohonneo4j.herokuapp.com/)
